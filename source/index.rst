Welcome to payler's documentation!
==================================

What is this?
-------------

Payler is an asyncio_-based Python application intended to provide a way of delaying message execution.

There are two main goals:

- Provide a way of moving payloads from one datasource to another
- Reduce load on targeted data transfer / store backends

.. _asyncio: https://docs.python.org/3/library/asyncio.html
